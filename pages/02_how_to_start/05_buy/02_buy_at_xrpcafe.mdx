# NFTを購入する　xrp.cafe

<a id="05_02_01"></a>
## 1. ログインする

xrp.cafeにアクセスしてログインします。  
日本語化・ログインする方法は [マーケットにログインする](/02_how_to_start/02_login) を参照。  
ページ右上にウォレットアイコンが表示されていればログイン完了です。
- [xrp.cafe](https://xrp.cafe/)


<a id="05_02_02"></a>
## 2. 販売されているNFTを探すには？

様々な方法でNFTを探したり分析することができます。

### [トップページ](https://xrp.cafe/) の検索フォームから検索

コレクション名やアイテム名を入力して検索できます。

### [トップページ](https://xrp.cafe/) **トップ**/**トレンド**

24時間の総取引量、24時間の販売数がそれぞれランキング表示されています。  
現在人気のコレクションを確認することができます。

### [トップページ](https://xrp.cafe/) **注目のコレクション**

運営の方が選ぶ注目コレクションです。

### [コレクションページ](https://xrp.cafe/collections)
現在xrp.cafeに登録されているコレクションを人気順、新着順で並び替えることができます。  
探しているコレクション名を入力して検索することもできます。

### [統計ページ](https://xrp.cafe/stats)
取引量、変動などNFTの項目ごとにコレクションを並べ替えることができます。

### [Infoページ](https://xrp.cafe/info)
現在Mint可能なコレクションの一覧を確認することができます。

## 3. 購入する

### 出品されているNFTを購入する

固定価格で出品されているNFTを購入します。

1. 「**今すぐ購入**」をクリック

2. Xamanアプリのフッターナビより取引履歴に移動

3. 取引履歴一覧からxrp.cafeの「**NFTオファーの作成**」リクエストを選択

 <img className="" src="/manual_pic/05_02_pic01.png" alt="05_02_pic01" />

4. リクエスト内容を確認してページの一番下から承認

 <img className="" src="/manual_pic/05_02_pic02.png" alt="05_02_pic02" />


 ### Mint形式のNFTを購入する

Mint形式で出品されているNFTを購入します。

 1. 「**mint**」をクリック

  <img className="solo" src="/manual_pic/05_02_pic05.png" alt="05_02_pic05" />

 2. 金額を確認し、Mint数を入力してXamanで承認
 
 3. 「**Claim your NFT**」をクリック  
 （またはウォレットページの**Offers**へ移動）
 
 4. 「**Claim NFT**」をクリックして承認  
 
 **Offers**ページの「**Select All**」で複数のNFTをまとめてClaimする場合、Cubyという未実装のツールを紹介するポップアップが表示されます。  
 「Continue」を押すと、Xamanの取引履歴一覧にxrp.cafeからの「**NFTオファーの受け入れ**」リクエストが届きます。  
該当するリクエストを全て承認すると購入完了です。  
 Mint後にClaimしないままでいると、NFTはウォレットに移動できず宙ぶらりんになってしまいます。  
 Mintしたら忘れずにClaimしましょう。
  

### オファーを出す

オファーを出すことができます。

1. 「**オファーを作成**」をクリック

2. 金額を入力して「オファーする」より手続きを進める

3. Xamanの取引履歴一覧からxrp.cafeの「**NFTオファーの作成**」リクエストを選択

4. リクエスト内容を確認してページの一番下から承認
